# 퍽 슬래시
## 기본
- 장르: 캐주얼(뱀서라이크)
- 그래픽: 검은 바탕 LED 외곽선 강조(플레이어 및 정보: 하얀색/적:붉은색/이펙트:푸른색/강조색: 초록색(힐), 노란색(출구))
- 플랫폼 : 웹, 모바일
- 조작 : 드래그 앤 클릭
- 목표: 5분이라는 제한시간동안 최하층의 보스를 쓰러뜨리는 것이 목표. 내려갈수록 더 많고 체력이 많은 적. 레벨과 층별로 강화를 얻음. 
## 게임 플레이
- 여러 액션을 통해 무기로 공격, 주인공이 퍽(하키)이라는 것을 이용한 빠른 전투. 뱀서 느낌이 아닌 궁수의 전설 느낌의 (기본+무기)의 심화 방향 강화(ex. 총이라면 리코셰, 두발 발사, 발사 속도, 관통/ 검이라면 방패, 관통, 주변에 검날, 등/분신, 충돌시 폭발, 반동 때마다 더 빨라짐 등등)
- 적을 찾아다니고 다가오기까지 기다리는 것이 아닌 적 안에 그냥 플레이어가 적진 한 가운데 떨어지는 방식으로 적이 무한정 리필이 되는 느낌. 최종 업그레이드에는 쓸어버리는 방식(이 아니면 게임오버를 유도? -> 난이도 있게) 
- 30초간 모든 적 처치 못 하면 게임오버 -> 10번의 전투동안 스킬 최적화 하는 것이 목표, 맨 밑에 층에는 보스+ 잡몹 최다로 존재.
### 액션 종류
- 본인 드래그 : 근접무기 일반 공격 + 대시
- 본인 홀드(고정/충전 이후 해제는 바깥으로 당기면 취소) : 근접무기 궁극기
- 바깥 드래그 : 원거리 무기 일반 공격(홀드하는 동안 지속 발사)
- 바깥 홀드(고정/충전 이후 해제는 본인으로 당기면 취소) : 원거리 무기 궁극기
### 프로토타입 제한(근접: 레이저검/원거리: 레이저총)
- 본인 드래그 : 부딪힐때마다 휘두르는 파티클 + 대시 + 마지막 공격에는 약간 슬로우모션
- 본인 홀드(고정/충전 이후 해제는 바깥으로 당기면 취소) : 다음 대쉬와 반동이 엄청나게 빨라짐
- 바깥 드래그 : 레이저 총(홀드하는 동안 뿅뿅 지속 발사)
- 바깥 홀드(고정/충전 이후 해제는 본인으로 당기면 취소) : 지속 레이저
### 적 종류(붉은색)
- 동그라미 : 근접 접근 유닛, 보통유닛 -> 대시공격시 튕겨나감
- 세모 : 원거리 공격 유닛, 속도 빠름, 체력약함  -> 대시 공격시 관통
- 네모 : 탱커 설정 유닛, 속도 느림, 아머가 있어서 대시 충돌시 끊김 -> 일정 데미지 입으면 갑옷이 부숴지고 동그라미로 바뀜.
- 악마: 보스, 체겨 많음 -> 대시 공격시 튕겨나감
### 오브젝트 종류
- 십자가(초록색): 체려회복
- 주먹(초록색): 공격력 강화
- 별(무지개): 퍽 하나
- 동그라미(출구/노란색): 탈출구, 모든 적을 처리하면 열림
### 퍽 종류
- 획득 시기: 레벨업 + 스테이지 하나 끝날 때 + 별 획득
- 총 강화 : 데미지+1, 한번에 쏘는 수+1, 관통+1, 벽에 튕기기+1, 연속발사+1 
- 검 강화 : 데미지+1, 튕길때마다 데미지+1, 대시 거리+1, 충돌처치마다 1회성 분신 소환+1
- 본체 강화 : 처치시 체력 흡수+1,  주위회전원반+1, 처치시 대상 폭발+1, 5초 안 맞으면 배리어 생성
